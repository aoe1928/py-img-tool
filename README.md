# Simple Batch Image Resizer 🖼️

Python製の画像一括リサイズ＆フォーマット変換ツールです。
フォルダごとドラッグ＆ドロップするだけで、指定したサイズと形式（JPEG/WebP）に一括変換します。

## 特徴 ✨

* **ドラッグ＆ドロップ対応**: ターミナルにファイルやフォルダを放り込むだけで動きます。
* **フォルダ一括処理**: フォルダを指定すると、中の画像をすべて変換します。
* **WebP対応**: JPEGだけでなく、軽量なWebP形式への変換も可能です。
* **上書き防止**: 同名のファイルがある場合、自動で `_1`, `_2` と連番を振って保存します。
* **ハイブリッド仕様**: 対話モードとコマンドライン引数の両方に対応しています。

## インストール 📦

Python 3がインストールされている必要があります。
